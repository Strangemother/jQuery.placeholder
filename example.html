jQuery.placeholder() Usage Examples

This will show you a simple example of the placeholder plugin. For the purposes of this example - the javascript placeholder will be 'J Placeholder' and the placeholder within the attribute will be 'type something'.

Simple Example $('#input').placeholder();

<input type="text" name="input" id="input" />

A placeholder attribute and no override specified in js $('#input1').placeholder('J Placeholder 1');>

<input type="text" name="input1" id="input1" placeholder='type something' />

With a placeholder attribute and with override specified $('#input1').placeholder('J Placeholder 2', true);

<input type="text" name="input2" id="input2" placeholder='type something' />

An example without a placeholder attribute.$('#input3').placeholder('J Placeholder 3');

<input type="text" name="input3" id="input3" />
